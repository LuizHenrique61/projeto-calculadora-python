# 🧮 Calculadora Simples em Python

Este é um projeto de **calculadora simples** feita em Python.  
Ela permite realizar as quatro operações matemáticas básicas: **soma, subtração, multiplicação e divisão**.

---

## 🎯 Objetivo
O objetivo deste projeto é praticar:
- Estruturas condicionais em Python
- Funções
- Entrada e saída de dados no terminal

---

## 🚀 Como executar
1. Certifique-se de ter o **Python** instalado (versão 3 ou superior).
